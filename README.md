BrewHaven – Luxury Coffee Shop Website
A modern and responsive coffee shop website developed as part of my internship project at DecodeLabs. The website features an elegant user interface, smooth navigation, and a premium design to showcase a luxury coffee brand.

Project Overview
BrewHaven is a luxury coffee shop website designed to provide users with an engaging and visually appealing browsing experience. The project focuses on responsive design, modern UI components, and smooth user interactions using the latest frontend technologies.

Features
Responsive design for desktop, tablet, and mobile devices
Modern and elegant user interface
Smooth scrolling and animations
Interactive website sections
Fast loading performance
Clean and well-structured code

Installation
Clone the repository:

git clone https://github.com/Eshal611/DecodeLabs-BrewHaven.git
Install dependencies:

npm install
Run the development server:

npm run dev
Build the project:

npm run build
Project Purpose
This project was developed during my internship at DecodeLabs to enhance my frontend development skills by building a modern, responsive, and production-ready web application using React and Vite.
